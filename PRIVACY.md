# Privacy Policy

English | [中文](#隐私政策)

Prompt Favorite is a local-first macOS utility. It does not collect, transmit, sell, or share personal data.

## Data Collection

Prompt Favorite does not collect analytics, telemetry, account information, contact information, identifiers, diagnostics, or usage data.

The app does not use a backend service and does not send your prompt text to any server.

## Local Files

Saved prompts are written only to the local folder you choose. The output files are normal Markdown files on your Mac.

If you choose a folder that is synced by another service, such as iCloud Drive, Dropbox, Git, or Obsidian Sync, that service may process the files according to its own privacy policy. Prompt Favorite does not control those services.

## Clipboard and Accessibility

To capture selected text from another app, Prompt Favorite asks macOS for Accessibility permission and temporarily sends a standard copy command to the frontmost app. It reads the copied text from the macOS pasteboard, then restores the previous clipboard content when possible.

This capture process happens locally on your Mac.

## Contact

For support, use the GitHub repository:

https://github.com/QQQingyu/prompt-favorite

# 隐私政策

[English](#privacy-policy) | 中文

Prompt Favorite 是一个本地优先的 macOS 工具。它不会收集、传输、出售或共享个人数据。

## 数据收集

Prompt Favorite 不收集分析数据、遥测数据、账号信息、联系人信息、标识符、诊断数据或使用数据。

App 不使用后端服务，也不会把你的 prompt 内容发送到任何服务器。

## 本地文件

收藏的 prompt 只会写入你选择的本地文件夹。输出内容是普通 Markdown 文件，保存在你的 Mac 上。

如果你选择的文件夹会被 iCloud Drive、Dropbox、Git、Obsidian Sync 等其他服务同步，这些服务可能会按照各自的隐私政策处理文件。Prompt Favorite 不控制这些服务。

## 剪贴板和辅助功能权限

为了从其他 App 捕获选中文本，Prompt Favorite 会请求 macOS 辅助功能权限，并向当前前台 App 临时发送一次标准复制命令。它会从 macOS 剪贴板读取复制出来的文本，并在可能的情况下恢复之前的剪贴板内容。

这个捕获流程只在你的 Mac 本地完成。

## 联系方式

如需支持，请使用 GitHub 仓库：
